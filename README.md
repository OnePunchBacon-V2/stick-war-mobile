# Stick War Legacy Mobile - React TypeScript

A fully responsive mobile web game based on Stick War Legacy with:
- 16:9 aspect ratio support
- Touch controls optimized for mobile
- Fullscreen API integration
- Landscape orientation lock
- 6 unique unit types with proper mechanics
- Tower defense gameplay
- Campaign progression

## Features

### Units
- **Swordwrath** (150g) - Fast melee fighter
- **Archidon** (300g) - Ranged archer
- **Spearton** (500g) - Tank with shield
- **Magikill** (1200g) - Mage with minions
- **Giant** (1500g) - Ultra tank unit
- **Miner** (150g) - Economy unit

### Game Modes
- Campaign progression
- Custom battles
- Endless mode

## Setup

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
